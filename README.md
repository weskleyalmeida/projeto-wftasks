# WF Tasks

WF Tasks é um sistema de gerenciamento de tarefas desenvolvido para ajudar na organização do dia a dia de forma simples, rápida e intuitiva.

## Funcionalidades

* Criar tarefas
* Editar tarefas
* Excluir tarefas
* Marcar tarefas como concluídas
* Cadastro e login de usuários
* Confirmação de conta por e-mail
* Interface responsiva
* Modo claro e escuro

## Tecnologias

* HTML
* CSS
* JavaScript

## Objetivo

Este projeto foi desenvolvido com o objetivo de praticar desenvolvimento web e criar uma aplicação funcional para gerenciamento de tarefas.
